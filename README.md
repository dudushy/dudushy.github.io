# [dudushy's portfolio](https://dudushy.github.io/)

---
## Prerequisites:
- [NodeJS v20.11.0](https://nodejs.org/download/release/v20.11.0/)

## How to install and run:
1. Install packages
    ```bash
    npm i
    ```
2. Run project
    ```bash
    npm start
    ```
3. Build project
    ```bash
    npm run build
    ```
4. Deploy project (gh-pages)
    ```bash
    npm run deploy
    ```
